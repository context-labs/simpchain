# Overview

Welcome to the official documentation for [Context](https://usecontext.io), an innovative AI-powered solution designed to help you build a chatbot that can efficiently answer questions based from various online resources. With Context, you can consolidate information from documentation websites, blogs, YouTube videos, and GitHub repositories, making it easier than ever to create a responsive and knowledgeable chatbot for your website, Discord, or Slack.

In this documentation, we'll provide you with an overview of Context, guide you through the quick start process, and introduce you to more advanced features and customization options.

Context is an early stage product, and we are gathering feedback as we go to improve the platform. If you run into any issues, or have suggestions or feedback, please email us at hello@usecontext.io.
